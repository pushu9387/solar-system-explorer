A 3D interactive model of our solar system built with Three.js.

## Features
- Realistic planetary orbits and sizes (not to scale for better visualization)
- Detailed information about each celestial body
- Guided tour mode
- Adjustable time scale
- Responsive design works on mobile and desktop

## How to Run
1. Clone this repository
2. Open `index.html` in your browser
3. Explore the solar system!

## Controls
- Click/tap planets to view information
- Use the UI panel to adjust settings
- Start the guided tour to learn about each planet

## Textures
Place planet texture files in the `public/textures` folder.

## Technologies Used
- Three.js
- Tween.js
- WebGL
